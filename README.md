# 3D Brain Visualization

Интерактивная 3D визуализация мозга из частиц с цифрами 0 и 1.

## Особенности

- 3D модель мозга из частиц
- Настраиваемые цвета для нулей и единиц
- Реалистичные тени с направленным освещением
- Фоновые частицы вокруг модели
- Интерактивное управление камерой

## Настройка GitHub Pages

1. Перейдите в Settings → Pages вашего репозитория
2. В разделе "Source" выберите "GitHub Actions"
3. Workflow автоматически запустится при каждом push в master/main

Проект будет доступен по адресу: `https://aalout.github.io/3D-brain/`

## Локальный запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

## Настройки

Все настройки доступны в объекте `colorSettings` и могут быть изменены из консоли браузера:

```javascript
// Цвета основной модели
colorSettings.color0.set(1.0, 1.0, 1.0); // Цвет нулей
colorSettings.color1.set(42/255, 241/255, 107/255); // Цвет единиц

// Цвета фоновых частиц
colorSettings.bgColor0.set(1.0, 1.0, 1.0); // Цвет фоновых нулей
colorSettings.bgColor1.set(42/255, 241/255, 107/255); // Цвет фоновых единиц

// Размеры
colorSettings.size0 = 1.0; // Размер нулей
colorSettings.size1 = 1.0; // Размер единиц

// Интенсивность теней
colorSettings.shadowIntensity = 0.75;

// Применить изменения
updateColors();
```
